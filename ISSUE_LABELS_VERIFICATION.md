## How to View Labels on GitHub

Labels should be visible on GitHub in several places:

1. **Issue List View:**
   - Go to: https://github.com/jessesep/claude-code-discord/issues
   - Labels appear as colored badges next to each issue title

2. **Individual Issue View:**
   - Click on any issue number
   - Labels appear at the top right of the issue, below the title

3. **Filter by Labels:**
   - Use the "Labels" filter dropdown on the issues page
   - Select a label to filter issues

4. **Label Management:**
   - Go to: https://github.com/jessesep/claude-code-discord/labels
   - View all available labels and their colors

---

## If Labels Don't Appear

If you don't see labels on GitHub:

1. **Refresh the page** - Sometimes GitHub needs a refresh to show new labels
2. **Check your view** - Make sure you're looking at the issues list, not pull requests
3. **Check permissions** - Ensure you have access to view the repository
4. **Try a different browser** - Clear cache or try incognito mode
